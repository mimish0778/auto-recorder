# Auto Take Recorder

ブラウザだけで動く、カウントダウン付き10秒動画収録ツールです。サーバー不要、`auto_recorder.html` を開くだけで動作します。

## 動作環境

- Chrome / Edge など、`getUserMedia` と `MediaRecorder` に対応したモダンブラウザ
- カメラ・マイクへのアクセス許可が必要
- PC・スマホどちらでも動作します

## 使い方（1テイクの流れ）

1. **Position** ボタンで `pos0`〜`pos4` のいずれかを選ぶ（ファイル名に使われるラベル）
2. **Start #** に保存するファイル番号を入力する（例：`4` → `pos2_004.mp4`）
3. **START** を押す
4. 画面中央に **3・2・1** のカウントダウンが表示される
5. カウントダウン終了と同時に **10秒間の録画** が自動的に始まる
6. 10秒経過すると **自動的に停止・保存** され、待機状態に戻る

**同じ番号のまま何度でも撮り直せます。** 番号は自動では増えません。次の番号に進みたいときは、**Start # を手動で書き換えて**からもう一度STARTを押してください。

## STOPボタンについて

STOPは「中断・キャンセル」専用です。

- カウントダウン中に押す → 録画は始まらず、キャンセルされる
- 10秒経過前（録画中）に押す → **そのテイクは破棄され、保存されない**
- **10秒を満了したテイクだけが保存されます。** 途中で失敗に気づいた場合はSTOPで即座に破棄できます

## 保存されるファイル

- ファイル名：`{position}_{番号3桁}.mp4`（例：`pos2_004.mp4`）
- 拡張子は、ブラウザが対応しているコーデックにより `mp4` または `webm` になります
- 解像度：**360×640**（縦長 9:16）固定。カメラの実解像度に関わらず、常にこのピクセルサイズで保存されます
- 保存先：ブラウザの標準ダウンロードフォルダ（このツール自体はフォルダ分けを行いません）

## 画面の見方

| 表示 | 意味 |
|---|---|
| 画面上部中央「FRONT camera / BACK camera」 | 現在使用中のカメラの種別 |
| 画面右上のカメラアイコン | 前面/背面カメラの切り替えボタン |
| 黄色い警告バッジ（Source resolution below target） | カメラの実解像度が360×640に届かず、映像を引き伸ばして保存していることを示す |
| 赤い「Recording」表示とタイマー | 録画中であることと経過秒数 |
| 画面下部のログ表示 | 現在の状態（カメラ準備完了・録画中・保存完了・破棄など）を文字で表示 |

## ロック仕様

録画中・カウントダウン中は、以下の操作ができないようにロックされます（誤操作でテイクが台無しになるのを防ぐため）。

- Position（撮影区間）の変更
- Start #（番号）の変更
- カメラの前面/背面切り替え

STOPを押すか、10秒経過して自動停止すると、ロックは解除されます。

## プレビューについて

- プレビュー映像は、実際に録画・保存される範囲と同じ縦長9:16の枠で表示されます（見えている範囲＝保存される範囲）
- 前面カメラ使用時、プレビューは見やすさのため左右反転（ミラー）表示されますが、これは表示上だけの処理です。実際に保存される動画データは、常にカメラが捉えた「実際の向き」（ミラーなし）で記録されます

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| 黄色い警告バッジが出る | カメラの実解像度が360×640未満のため、映像を引き伸ばして保存している状態。録画自体は問題なく行われる |
| カメラ切替ボタンが反応しない | 録画中・カウントダウン中は切替不可（仕様どおり）。STOPを押してから切り替える |
| Position/Start #が変更できない | 録画中・カウントダウン中はロックされる仕様。STOPを押すと解除される |
| カメラの映像が出ない/エラーが出る | ブラウザのカメラ・マイク許可設定を確認する。画面下部のログにエラーメッセージが表示される |
